    CSS Box Model Example

    the CSS box model is essentially a box that wraps around every html element it consist of:

    • Margins: outermost layer, creates space between elements
    • Borders: wraps around the padding and content
    • Padding: space between the content and the border
    • Content: innermost layer, where text and images appear
    This box demonstrates the CSS box model.

    In this example, the box has a width of 200px, padding of 20px, a border of 5px, and a margin of 10px. The background color is light yellow.
